## 一、系统介绍
该系统使用PHP+MySQL开发，运行于Web端，主要用于各类学科测评成绩的录入、统计和分析，适用于各类学校、机构，用于管理学生测评数据，系统部署简单，跨平台兼容，操作逻辑清晰，容易上手，且有详细操作日志记录及关键操作区域水印等安全措施，确保录入的成绩数据不外泄，确保数据安全。

## 二、系统功能介绍

**（一）账号多设备登录限制说明**

	除班主任账号外，其他账号都不允许单一账号在多设备中同时登录使用，所以使用完请务必记得点击右上角的“退出”按钮注销登录，否则其他用户在30分钟内无法使用该账号再次登录系统。
  
**（二）账号密码修改功能说明**

	登录账号后，可以点击右上角的用户信息修改姓名及密码。（班主任、数据录入账号不支持该功能）
  
**（三）账号类型权限说明**

	系统共有四类不同角色的账号，权限功能各不相同。
    ①	管理员角色：拥有系统所有权限及测评数据录入模块的查看权限，只有管理员账号才有“账号管理”和“日志管理”这两个模块的访问权限。
    ②	教导处/教务处角色：拥有除“账号管理”和“日志管理”这两个模块的之外的所有访问权限及测评数据录入模块的查看权限。
    ③	班主任角色：拥有学生信息管理模块的权限。
    ④	阅卷老师角色：拥有测评数据录入模块的录入编辑权限及数据看板中看班级、看年级、看语数三个模块的访问权限。
    
**（四）系统使用流程**

	①管理员角色先创建相关角色的账号；
	②由管理员角色或教导处角色先到“项目管理”模块创建项目；
	③由管理员角色或教导处角色到“班级管理”模块创建年级和班级结构；
	④由管理员角色或教导处角色到“学科管理”模块创建相关学科；
	⑤由管理员角色对相关账号赋予访问/编辑权限；
	⑥班主任角色录入本班学生信息（也可由管理员/教导处统一录入）；
	⑦阅卷老师录入本账号权限下的学科成绩数据，并查阅/抄录数据看板中的统计数据；
	⑧教导处下载相关学科的成绩及统计数据。
  
**（五）系统各模块功能介绍**

**1.项目数据看板**

	登录系统后，首先看到的是项目数据看板，在该页面中可以直观掌握各班的任务完成情况。
 
**2.系统设置-项目管理**

	在相同年级、学科中需要进行多次不同时期的测评数据录入统计，则需要创建多个不同的项目。如：期中测试，那么就需要创建一个新项目用来管理期中测试的学生、学科和数据；期末测试，则需要另外新建一个项目来管理期末测试的学生、学科和数据。
	只有在项目状态为“可用”状态下时，项目中的班级、学生、学科及相关数据才能被编辑（下载数据不受此限制）。在新建项目时，可以选择同步指定项目中的班级、学生和学科信息数据（不会同步成绩数据）。
 
**3.系统设置-班级管理**

	创建新项目时，如果没有同步旧项目的班级数据，则需要手动到班级设置中创建年级、添加班级。
	创建年级的时候，需要输入年级名称和年级代码，年级代码不允许重复（支持字母、数字格式）。删除年级时，需要先删除该年级下的所有班级方可删除该年级。
	创建班级的时候，需要先选择年级，再设置班级名称和班级代码，班级名称和班级代码不允许重复。删除班级时，需要先到“学生信息管理”界面删除该班级下的所有学生方可删除该班级。（学生管理中支持批量删除）
  
**4.系统设置-学科管理**

	可以添加和设置测评的学科及学科的分数线划分以及所适用的年级。
	注1：学科管理中的“成绩拆分”功能适用于如高年级语文基础成绩和作文成绩分开统计的情况。
	注2：如果学科所关联的班级已有成绩数据，则不允许删除或编辑修改成绩拆分状态、适用年级。
 
**5.系统设置-账号管理**

	账号管理页面只有管理员角色有权限访问，此处可以添加、删除、编辑系统中所有账号。
 
**6.系统设置-日志管理**

	日志管理页面只有管理员角色有权限访问，此处主要记录了所有账号角色的关键操作，方便溯源。
 

**7.学生信息录入**

	使用分发的对应年级的班主任账号登录，点击顶部菜单栏的“学生信息录入”进入学生信息录入界面：
	不同账号赋予的权限不同，请从年级、班级列表中选择授权范围内相应的年级、班级，点击右侧的“批量导入”按钮，打开对话框，然后将本班所有学生姓名复制粘贴进去（一行一个名字），根据需求选择是否自动排序，再点击“导入”按钮即可完成。
	在选择相应的班级后，页面中会有“重新排序”按钮，点击后即可对学生的学号进行重新编排。
	注：对学生学号进行编排不会影响学生已有的成绩数据，学号不允许留空跳过。所以如果你班级中实际学生的号数中间有跳过的，也请你按照你班级中学生的排序按顺序编排下去。
 
**8.测评数据录入**

	使用分发的对应年级学科的登分账号登录，点击顶部菜单栏的“测评数据录入”进入测评数据录入界面：
	不同账号赋予的学科和年级权限不同，请从年级、学科、班级中选择授权范围内的对象，点击右侧的“录入数据”按钮，进入数据录入状态：

**9.数据看板**

	系统提供班级、年级、语数学科三类数据统计面板，完成相应学科的班级数据录入后，即可实时生成相应的数据统计，数据查看面板只对测评数据录入员及教导处/教务处权限的账号开放。
	通过“看班级”面板，除了可以查看班级数据分布，还能在下方的“学生列表”中快速了解班级成绩排名。
	通过“看年级”面板，也可以快速了解年级成绩排名
 
**10.数据看板-数据下载**

	数据下载只针对教导处/教务处/管理员角色账号开放，提供单个学科的年级数据下载及指定年级的语数学科整合数据下载。
	点击对应卡片即可选择项目、年级、学科及下载参数进行数据的下载和导出
